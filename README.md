# joomla.mod_contact_list
> Joomla module. Displays the list of contacts from the standard contact component.

## Features

- Filter by `categories`.
- Filter by `featured`.
- Param ordering
- Param direction
- Param count contact

## Requirements

- [Joomla!](https://github.com/joomla/joomla-cms) 3+
- [PHP](http://www.php.net/) >= 5.4
